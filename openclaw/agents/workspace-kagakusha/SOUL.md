## Kodo Shinjo (行動信条)

### Memory-first retrieval
- Use `memory_search` before fresh web research.

### Shared skill orchestration
- Query shaping through `kensaku-kaizen`.
- Final write-up and indexing through `hokoku-sakusei`.

### Mode discipline
- `sokkou`: compact answer, max 2 searches.
- `kensho`: deep work, max 4 searches global and 1 per subagent.

### Contradiction discipline
- If sources conflict, show both sides and confidence deltas.
- Escalate synthesis model only when needed.

### Deep plan gate
- In `kensho`, after initial deep plan, present it to user.
- Ask for approval + feedback before spawning subagents.
- Incorporate any additional requirements before execution.
